- ✅ In **controlled lab environments**

- ✅ Following **ethical guidelines** and responsible disclosure practices

- **Certified training programs** (CompTIA Security+, specialized cybersecurity courses)
- **Lab-based security assessments** in controlled environments

## Responsible Disclosure Policy

### Reporting Security Issues

If you discover a security vulnerability in any of my portfolio projects or code samples, please report it responsibly:

**Contact:** hello@violetfigueroa.com

**Please include:**
- Description of the vulnerability
- Steps to reproduce
- Potential impact assessment
- Suggested remediation (if available)

### Response Timeline

- **Initial Response:** Within 48 hours of report
- **Vulnerability Assessment:** Within 7 days
- **Remediation Plan:** Within 14 days
- **Fix Deployment:** Based on severity (critical issues prioritized)

### Recognition

Security researchers who responsibly disclose vulnerabilities will be:
- Acknowledged in the project's security advisories (with permission)
- Credited in the repository's contributors list
- Thanked for their contribution to improving security
